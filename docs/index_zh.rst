RecIS 文档
=================

.. note::
   **Language Selection**
   
   - English Documentation: :doc:`index`

RecIS (Recommendation Intelligence System) 是一个统一架构的深度学习框架，专为超大规模稀疏 + 多模态 + 大语言模型而设计，基于 PyTorch 开源生态构建。

快速开始
--------

1. **安装**: 按照 :doc:`installation` 指南设置 RecIS
2. **快速入门**: 尝试 :doc:`quickstart` 教程构建您的第一个模型
3. **示例**: 浏览 :doc:`examples/index` 查看详细用例
4. **API 参考**: 查看 :doc:`api/index` 获取完整的 API 文档

核心特性
--------

- **超大规模支持**: 面向超大规模稀疏模型统一架构
- **高性能**: 基于 PyTorch 开源生态，支持稀疏模型分片优化
- **易用性**: 提供简洁的 API 设计，支持原生计算优化技术
- **兼容性**: 与现有 PyTorch 生态完全兼容
